# Glossary

Selected keywords. Full list: [Glossarium (Ch. XV)](https://adeptusprogus.github.io/vox-gothica/15-glossary.html)

| Keyword | Latin | Role |
|---------|-------|------|
| `AVE OMNISSIAH` | Hail Omnissiah | File seal (required) |
| `CANTICUM` | song | Program entry |
| `FABRICA` | workshop | Infrastructure manifest |
| `LITANIA` | litany | Library module |
| `RITUS` | rite | Function |
| `SCHEMA` | form | Record type |
| `DECLARO` | I declare | Mutable variable |
| `SANCTUM` | holy | Immutable binding |
| `FIAT` | let it be | Assignment |
| `VOCIFERO` | I cry out | Print stdout |
| `SUSURRO` | I whisper | Print stderr |
| `SI` / `ALITER` | if / else | Conditional |
| `DUM` | while | Loop |
| `PRO OMNI … IN … AGE` | for each … in … | Iteration |
| `TEMPTA` | I attempt | Try |
| `HERESIS` / `HERESIM` | heresy | Catch / raise error |
| `INVOCO` | I invoke | Import |
| `REDDO` | I return | Return value |
| `NUMQUAM FINIS` | never end | Block terminator |
| `EXSTRUATUR` | let it be built | Resource (Fabrica) |
| `FOEDUS` | covenant | Provider (Fabrica) |
| `POSTULO` | I demand | Input variable (Fabrica) |
| `PROFITEOR` | I proclaim | Output (Fabrica) |
| `NIHIL` | nothing | Null |
