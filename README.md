# Vox Gothica

**A High Gothic programming language for canticles and fabricae** — an esolang in the
tradition of the Adeptus Mechanicus, with a real Python toolchain (`gothica`) and
Terraform-backed *fabricae*.

[![License: GPL v3](https://img.shields.io/badge/License-GPLv3-blue.svg)](LICENSE)
[![Python](https://img.shields.io/badge/python-3.10+-blue.svg)](https://www.python.org/downloads/)
[![Docs](https://img.shields.io/badge/docs-Codex%20Vox%20Gothica-darkred)](https://adeptusprogus.github.io/vox-gothica/)

> *From the weakness of the flesh — the Machine delivers us.*

Vox Gothica is a programming language whose source reads as the pseudo-Latin liturgy of the Adeptus Mechanicus. One syntax serves two rites:

- **Canticles** — general-purpose programs: rites (functions), schemas (records), heresy handling (errors), modules, and packages (*litaniae*).
- **Fabricae** — infrastructure manifests that transpile to Terraform JSON. Chanting genuinely raises cloud machines.

```vg
AVE OMNISSIAH.
CANTICUM "salutatio".

VOCIFERO "Ave, Imperium!"
```

```console
$ gothica invoco salutatio.vg
Ave, Imperium!
++ The rite concludes. The Machine Spirit is appeased. ++
```

## Quick start

```console
git clone https://github.com/adeptusprogus/vox-gothica.git
cd vox-gothica/vox-gothica
./install.sh
gothica invoco exempla/salutatio.vg
```

**Requirements:** Python ≥ 3.10, zero runtime dependencies.

## Toolchain (`gothica`)

| Command | Action |
|---|---|
| `gothica invoco file.vg` | Run a CANTICUM |
| `gothica proba --dir probationes` | Run `*_proba.vg` test litanies |
| `gothica scribe-solum fabrica.vg` | Emit Terraform JSON only |
| `gothica auguro fabrica.vg` | Plan (`--exi-si-mutatio` for drift CI) |
| `gothica consecro fabrica.vg` | Apply — type `FIAT` (`--fiat` to skip) |
| `gothica exterminatus fabrica.vg` | Destroy — type `EXTERMINATUS` |
| `gothica versio` | Version |

Global flags: `--silens`, `--profanum` (JSON diagnostics for CI), `-postulatum k=v`.

See [vox-gothica/README.md](vox-gothica/README.md) for install options (pip, zipapp, PyInstaller, Docker).
