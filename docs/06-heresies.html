06 — The Codex Hereticus (Error Taxonomy)

This chapter is fully normative. For every condition below, a conforming implementation must raise exactly the stated genus. This is the doctrinal answer to the question "what counts as heresy?"

1. The four orders of failure

OrderLatinWhen detectedCatchable by TEMPTA?Process exit code
ProfanationPROFANATIOparse / load time, before any code runsno — the scroll itself is corruptII
Minor HeresyHERESIS MINORISruntimeyesI (if uncaught)
Major HeresyHERESIS MAIORISruntime / consecrationno — execution is condemnedI
Wrath of the MachineIRA MACHINAETerraform subprocess failureno (occurs outside the interpreter)III

Package-management failures form a fifth, tool-level order, Excommunication (EXCOMMUNICATIO) — they occur in gothica adfero/offero, never during program execution. Exit code IV.

Doctrine of catchability: a heresy is catchable iff the program could plausibly recover and the interpreter's own invariants are intact. Structural corruption (duplicate resource names, import cycles, exhausted memory) is never catchable — recovery code would run inside a lie.

A Major Heresy unwinds the entire stack, ignoring TEMPTA guards; DENIQUE blocks do still run (resources must be released even as the ship burns).

6. TEMPTA semantics (normative)


TEMPTA <body>
SI HERESIS (nomen) <catch>        ++ optional
DENIQUE <finally>                 ++ optional; at least one of the two must appear
FINIS TEMPTA
  1. Body runs. If no heresy: catch is skipped, finally runs.
  2. Minor heresy in body: body aborts at the raise point, catch runs with the heresy bound, then finally.
  3. Heresy in catch: finally runs, then the new heresy propagates (the original is attached as .origo, a nullable HERESIS field).
  4. Heresy in finally: it wins; anything propagating is attached as its .origo.
  5. REDDO/RUMPO/PERGO inside body or catch: finally runs first, then the jump completes.
  6. Selective catching is by inspection — SI delictum.genus == "clavis_ignota" TUNC ... ALITER PROCLAMO HERESIM delictum re-raises. (PROCLAMO HERESIM <HERESIS value> re-raises preserving all fields; this is the only way a HERESIS value may be "constructed".)
  7. Major heresies ignore catch arms entirely (§4) but honor finally arms.

7. Raising


PROCLAMO HERESIM "The divisor is naught!"

9. Wrath of the Machine (infrastructure)

Raised when terraform exits non-zero during auguro/consecro/exterminatus. The banner is themed; Terraform's own stderr is reproduced verbatim below it, untruncated — obscuring the true error is itself heresy (of the toolsmith). Exit code III.

10. Canonical diagnostic format

See 12-cli-and-diagnostics §2. Every heresy/profanation surfaces with: banner, genus, file + Roman VERSUS, message, source excerpt with caret, hint (if the genus has one), and stack trace for runtime heresies.
