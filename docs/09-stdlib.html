09 — The Machine-Cult (Standard Library)

Stdlib names always win import resolution (07-modules §3) and cannot be shadowed by litaniae. Every heresy a stdlib rite can raise is listed; raising anything not listed is non-conformant.

2. mathematica

radix (sqrt; negative → argumentum_pravum), potentia(b, e), modulus (abs), minimus(ordo) / maximus(ordo) (empty → argumentum_pravum), rotundo(f) (half-to-even), solum(f) / tectum(f) (floor/ceil), fractio_integra(f) -> NUMERUS (NaN/inf → conversio_impossibilis), sinus cosinus tangens logarithmus exponens; constants PI, E, NAN, INFINITUM, MAXIMUS_NUMERUS, MINIMUS_NUMERUS.

3. scriptura

maiuscula minuscula (case), divide(s, sep) -> ORDO, coniunge(ordo, sep), substitue(s, vetus, novum), incipit desinit continet -> VERITAS, purga (trim), imple_sinistra(s, n, c) (pad), inveni(s, sub) -> NUMERUS (−1 if absent — the one sanctioned −1, documented loudly), mensura_octetorum, ad_octetos / ex_octetis (UTF-8 ↔ ORDO[NUMERUS]; invalid bytes → forma_prava).

4. ordo_opera (list rites — all take a RITUS and return new lists)

transmuta(ordo, f) (map), excerne(ordo, f) (filter — "separate the pure"), redige(ordo, f, initium) (fold), ordina(ordo) / ordina_per(ordo, f) (sort, stable; mixed types → typus_profanus), inverte(ordo), summa(ordo), quisque(ordo, f) / ullus(ordo, f) (all/any), compone(a, b) -> ORDO[ORDO] (zip, stops at shorter).

5. tempus

nunc() (epoch ms), nunc_monotonicum() (monotonic ns, for timing), data_scriptum(epocha_ms, forma) / data_lege(s, forma) (strftime-style; bad input → forma_prava). All UTC; timezones are Warp phenomena deferred to litania tempus_extra.

6. archivum (files)

lege(via) -> SCRIPTUM, lege_octetos(via) -> ORDO[NUMERUS], scribe(via, textum), attinge(via, textum) (append), existit(via) -> VERITAS, dele(via), enumera(via) -> ORDO[SCRIPTUM] (dir listing), crea_directorium(via). Heresies: archivum_ignotum, archivum_clausum. Paths use / on all hosts; the runtime translates.

7. imperium (process)

argumenta() -> ORDO[SCRIPTUM], ambitus(nomen) -> SCRIPTUM (unset env var → clavis_ignota; use ambitus_aut(nomen, defectus)), exi(codex), impera(mandatum : ORDO[SCRIPTUM]) -> TABULA (run subprocess; returns {"codex": N, "exitus": "...", "error": "..."}; spawn failure → archivum_ignotum).

8. machina_cogitans (HTTP)

pete(url) -> SCRIPTUM (GET), pete_plene(petitio : TABULA) -> TABULA (full request: method/headers/body; returns status/headers/body), mitte(url, corpus) -> SCRIPTUM (POST). Heresies: retis_fractum (transport), responsum_pravum (status ≥ 400; .gradus carries it). Timeout default XXX s, override via pete_plene.

9. codex_json

lege(s) -> valor (JSON → TABULA/ORDO/values; null → NIHIL — quarantine rules 02-types §7 apply on use; malformed → forma_prava), scribe(valor) -> SCRIPTUM, scribe_ornatum(valor) (pretty, 4-space). Schema instances serialize as objects; rites are conversio_impossibilis.

10. fortuna (randomness)

inter(a, b) (uniform inclusive NUMERUS), fractio_fortunae() ([0,1)), elige_sortem(ordo) (random element; empty → argumentum_pravum), misce(ordo) (shuffle in place), semen(n) (seed; unseeded = OS entropy). (The dice are blessed but honest.)

11. notarius (logging)

nota(gradus, nuntius) with gradus ∈ "debug" "info" "monitum" "error"; convenience debug/info/monitum/error(nuntius). Output to stderr: [GRADUS] tempus — nuntius. Level via GOTHICA_NOTARIUS env var or notarius.gradus_pone(g). Structured JSON mode: notarius.modus_json(VERUM).

12. probatio (testing — normative)

Test files: probationes/*_proba.vg, mode LITANIA, run by gothica proba. Every exported zero-argument rite whose name starts with proba_ is a test case.


AVE OMNISSIAH.
LITANIA "mathematica_proba".

INVOCO adfirma, adfirma_aequalia, adfirma_heresim EX "probatio"
INVOCO "fons/logistics"

RITUS proba_summa_recta () -> NIHIL
    adfirma_aequalia(logistics.summa_tributi(C), CX, "tithe of C must be CX")
FINIS RITUS

RITUS proba_divisio_nihili_capta () -> NIHIL
    adfirma_heresim("divisio_nihili", RITUS () -> NIHIL
        DECLARO x : NUMERUS = I / N
    FINIS RITUS)
FINIS RITUS

API: adfirma(cond, nuntius), adfirma_aequalia(actus, exspectatus, nuntius), adfirma_heresim(genus, ritus) (fails unless the rite raises exactly that genus), praetermitte(nuntius) (skip). A failed assertion raises genus probatio_fracta (Minor, catchable — the runner catches it). Runner output: one line per test, SANCTUS (pass) / HERETICUS (fail) / PRAETERMISSUS (skip); summary and exit I on any failure. gothica proba --unus proba_summa_recta runs one; --ordinatio fortuita randomizes order (seed printed).
